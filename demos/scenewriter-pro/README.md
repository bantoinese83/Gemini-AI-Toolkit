A Vite TypeScript React application that generates professional video scripts and production plans using AI.

## Features

- **AI Script Generation**: Create detailed video scripts with scenes, dialogue, and timing
- **Production Planning**: Generate shot lists, props, talent, and location requirements
- **Multi-Platform Optimization**: Tailored content for YouTube, TikTok, Instagram, etc.
- **SEO Metadata**: Automatic keyword and hashtag generation
- **Voiceover Scripts**: Professional narration scripts included
- **Sample Topics**: Pre-loaded examples for quick testing

## Concept

SceneWriter Pro helps content creators, marketers, and production teams generate professional video scripts automatically. Instead of starting from scratch, users input their topic and requirements, and AI generates:

1. **Complete video scripts** with scene breakdowns and timing
2. **Production plans** with shots, props, and crew requirements
3. **Audio scripts** for voiceover and music suggestions
4. **SEO optimization** with keywords and hashtags
5. **Platform-specific** content tailored for different audiences

## Tech Stack

- **Frontend**: Vite + React 18 with TypeScript
- **UI**: Tailwind CSS with modern React components
- **AI**: Gemini AI Toolkit (gemini-ai-toolkit)
- **Build**: Vite for fast development and building
- **Styling**: Responsive design with utility-first CSS

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up environment variables:**
   Create a `.env` file in the root directory:
   ```
   VITE_GEMINI_API_KEY=your-api-key-here
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:5173](http://localhost:5173)

## Demo Flow

1. **Choose Sample Topic**: Click any pre-loaded topic for instant setup
2. **Or Enter Custom Topic**: Input your own video concept
3. **Configure Parameters**: Set duration, style, audience, and platform
4. **Generate Script**: AI creates complete script and production plan
5. **Review Results**: Get script breakdown, production details, and SEO metadata
